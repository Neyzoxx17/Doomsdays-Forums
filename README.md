# Doomsday Forums

Forum de discussion moderne avec authentification Firebase et rôles utilisateurs.

## Fonctionnalités

- Authentification Firebase
- Système de rôles (Admin, VIP, Modérateur, Membre)
- Chat en temps réel
- Avatars personnalisés
- Panneau d'administration
- Design moderne et responsive

## Déploiement

Ce site est déployé sur GitHub Pages : https://neyzoxx17.github.io/Doomsdays-Forums/

## Configuration

- Firebase Realtime Database pour le stockage
- Firebase Authentication pour l'authentification
- Design responsive avec animations CSS
